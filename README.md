# EigoLock（英語学習アプリ・プロトタイプ）

「スマホ（対象SNS）を開くたびに英語で1問。答えると解除」で英語アウトプットを習慣化するアプリの**Webプロトタイプ**です。
デザインは同じGitHubフォルダ内 `yumekana-portal` のトンマナ（ミント×ティール／グラスモーフィズム）を踏襲しています。

> ⚠️ これはデザイン・UX確認用のプロトタイプです。AI添削は**サンプル（モック）**、データは端末内 `localStorage` のみに保存されます。

## 動かし方

いずれかの方法で：

- `index.html` をブラウザで直接開く、または
- 付属の静的サーバーを起動: `node server.js` → http://localhost:8123 を開く

## iPhoneにアプリとして入れる（PWA・App Store不要）

本アプリはPWA対応済み（`manifest.json`＋`sw.js`＋アイコン）。**Safariで開く→共有→「ホーム画面に追加」**で、独自アイコン・全画面のアプリとして起動します。

- **方法A（おすすめ・安定／公開済み）**: **https://ito-ops.github.io/eigolock/** をiPhoneのSafariで開く → 共有 → 「ホーム画面に追加」。HTTPSなのでオフライン（Service Worker）も有効。
  - リポジトリ: https://github.com/ito-ops/eigolock （`main` ブランチへの push で Pages が自動再ビルド）
- **方法B（同じWi-Fiで今すぐ）**: Macで `node server.js` を起動 → iPhoneのSafariで `http://<MacのIP>:8123`（例: `http://172.30.5.21:8123`、IPは `ipconfig getifaddr en0` で確認）→ 「ホーム画面に追加」。※httpのためオフライン機能（SW）は無効、オンラインでは動作。

## 実装している体験（要件との対応）

- **初回設定画面** … レベル／興味テーマ／学びたい英語の地域／ロック対象アプリ／「あとで」ON-OFF／通知
- **ホーム** … 連続日数・今日の回答・平均スコア、対象アプリ（タップでロック体験）、弱点TOP3
- **ロック画面（出題）** … 対象アプリを開くと英語1問。地域イディオムのワンポイントを定期表示。「あとで」は設定で表示
- **AI添削（モック）** … ①修正後の自然な英文（差分ハイライト）②文法・単語の補足 ③日本語のワンポイント解説 ④ネイティブの言い回し ⑤地域イディオム ⑥語彙＋スコア
- **復習** … 過去の出題・回答・添削の一覧／検索／カテゴリ絞り込み
- **単語・表現帳** … 添削から保存した語彙をフラッシュカードで復習
- **進捗** … 連続日数・累計・平均スコア・推定レベル・スコア推移・テーマ別回答数
- **設定** … 後からいつでも変更／データ初期化

## 本番に向けて（次フェーズ）

- **Gemini接続**: `index.html` 内の `fetchFeedback()` を、サーバーレス関数（プロキシ）経由でGeminiを呼ぶ実装に差し替え。**APIキーはクライアントに置かず**サーバ側の環境変数で保持。
- **iOS実機化**: ローカル通知＋Screen Time（Family Controls）でアプリ起動をゲートする"集中モード"として実装。
